起動方法
	FF [path1] [path2]
		path1 : フランク１のパス（省略時はカレント）
		path2 : フランク２のパス（省略時は空）
	<Tab>

フランクについて
	フランク１ : 基本ファイラ
	フランク２ : サブファイラ、もしくは簡易テキストヴューワ
	フランク３ : 選択中エントリの表示、編集
	
基本操作
	key map
		操作内容
		対応モード
			n : normal
			v : visual
			p : point
		対応エントリ
			d : directory
			f : file
			g : global（エントリに対するコマンドではない）
		対応フランク
	
	ファイル編集
		e
			カーソル下、もしくは選択範囲内のファイルを開く（ディレクトリは無視）
			n, v
			f
			1, 2
		e
			ポイントをつけた順で開く（ディレクトリは無視）
			p
			f
			1, 2

	ディレクトリ移動
		h
			ひとつ上のディレクトリに移動
			n
			g
			1, 2
		l
			ひとつ前のディレクトリに移動
			n
			g
			1, 2
		cd
			カーソル下のディレクトリに移動
			n
			d
			1, 2

	折りたたみ
		fh
			カーソル下のエントリを畳む
			n
			d
			1, 2
		fl
			カーソル下のエントリを開く
			n
			d
			1, 2
		fH
			カーソル以下のエントリを全て畳む
			n
			d
			1, 2
		fL
			カーソル以下のエントリを全て開く
			n
			d
			1, 2
		FH
			全てのエントリを畳む
			n
			g
			1, 2
		FL
			全てのエントリを開く
			n
			g
			1, 2

	ポイント
		p
			カーソル下、もしくは選択範囲内のエントリにポイントをつける
			n, v
			g
			1, 2
		P
			カーソル下、もしくは選択範囲内のエントリのポイントをはずす
			n, v
			g
			1, 2

	フランク３を用いるファイル操作
		cp
			コピー
			n, v, p
			d, f
			1
		mv
			移動
			n, v, p
			d, f
			1
		rm
			削除
			n, v, p
			d, f
			1
		re
			リネーム
			n, v, p
			d, f
			1
		mk
			ディレクトリ作成
			n
			d
			1
		to

			n
			d
			1

	フランク３を用いないファイル操作
		u
			取り消し
			n
			g
			1, 2
		U
			やり直し
			n
			g
			1, 2
		o
			ディレクトリを開く
			n
			d
			1, 2
		a
			関連アプリケーションで開く
			n, v, p
			f
			1, 2
		q
			クイックルックで開く
			n
			f
			1, 2
		de
			フランク２で詳細情報を見る
			n
			f
			1
		rd
			フランク２で簡易ビューワで開く
			n
			f
			1
		<CR>
			フランク３を用いるファイル操作の決定
			n

			1, 2
