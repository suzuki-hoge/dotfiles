起動方法
	FF [path1] [path2]
		path1 : フランク１のパス（省略時はカレント）
		path2 : フランク２のパス（省略時は空）
	<Tab>

フランクについて
	フランク１ : 基本ファイラ
	フランク２ : サブファイラ、もしくは簡易テキストヴューワ
	フランク３ : 選択中エントリの表示、編集
	
基本操作
	key map
		操作内容
		対応モード
			n : normal
			v : visual
			p : point
		対応エントリ
			d : directory
			f : file
			g : global（エントリに対するコマンドではない）
		対応フランク
	
	ファイル編集

	ポイント
